## 🔧 How to Compile & Run

**Compile:**
```bash
gcc filename.c -o output
```

**Run:**
```bash
./output        # Linux/Mac
output.exe      # Windows
```

**Example:**
```bash
gcc Hello.c -o hello
./hello
```

> Requires **GCC compiler**. On Windows use [MinGW](https://www.mingw-w64.org/) or [TDM-GCC](https://jmeubank.github.io/tdm-gcc/).

## 💡 Key Concepts Demonstrated

- **Pointers** — pointer arithmetic, void pointers, NULL pointers, array pointers
- **Structures** — nested structures, array of structures, structure pointers
- **Dynamic Memory** — `malloc()`, `calloc()`, `realloc()`, `free()`
- **File I/O** — `fopen()`, `fclose()`, `fprintf()`, `fscanf()`, `fgets()`
- **Preprocessor** — `#define`, `#include`, `#ifdef`, macros
- **Storage Classes** — `auto`, `static`, `register`, `extern`
- **String Operations** — `strlen()`, `strcpy()`, `strcmp()`, `strcat()`
- **Recursion** — recursive functions and problem solving
